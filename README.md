# Трекер прочитанных книг (Book Tracker)

**Автор:** Татарников Иван Алексеевич

---

## Описание

Десктопное GUI-приложение для управления списком прочитанных книг.
Программа позволяет добавлять, фильтровать, удалять книги и сохранять их в JSON-файл.

**Важная особенность:** использует **только стандартную библиотеку Python** (Tkinter, json, os).
Не требует установки дополнительных пакетов.

---

## Функциональные возможности

- ✅ Добавление книг (название, автор, жанр, количество страниц)
- ✅ Фильтрация по жанру
- ✅ Фильтрация по количеству страниц (например, показать книги, где страниц > 200)
- ✅ Подсчёт статистики (общее количество книг и страниц)
- ✅ Удаление записей
- ✅ Автоматическое сохранение в JSON
- ✅ Автоматическая загрузка при запуске
- ✅ Валидация ввода (поля не пустые, страницы — положительное число)

---
